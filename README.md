# Flutter x ERPNext Integration

Satu aplikasi mudah alih yang dibina menggunakan **Flutter** untuk berinteraksi dengan **ERPNext** melalui REST API. Projek ini membolehkan capaian data dan pengurusan proses perniagaan terus dari peranti pintar.

## 🚀 Ciri-ciri Utama
* **Integrasi REST API**: Hubungan terus dengan pelayan ERPNext.
* **Pengurusan Data**: Keupayaan untuk membaca dan menghantar data ke pelayan.
* **Seni Bina Modular**: Kod yang disusun mengikut komponen untuk memudahkan penyelenggaraan.

## 🛠️ Prasyarat
Sebelum memulakan, pastikan anda mempunyai:
* [Flutter SDK](https://docs.flutter.dev/get-started/install) dipasang di mesin anda.
* Akaun atau pelayan **ERPNext** yang aktif.
* Kebenaran API (API Key & API Secret) bagi pengguna yang berkaitan di ERPNext.

## ⚙️ Cara Penggunaan & Konfigurasi

Untuk menyambungkan aplikasi ini ke pelayan ERPNext anda sendiri, ikuti langkah-langkah di bawah:

### 1. Klon Repositori
```bash
git clone [https://github.com/iyon09/FlutterxERPNext.git](https://github.com/iyon09/FlutterxERPNext.git)
cd FlutterxERPNext
